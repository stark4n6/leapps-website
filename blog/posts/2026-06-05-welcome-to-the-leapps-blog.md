# Welcome to the LEAPPs Blog

After years of building tools and sharing them through GitHub releases and social media, we're launching a dedicated blog for the LEAPPs project — a place to go deeper.

## What to expect

- **Artifact deep dives** — detailed breakdowns of specific iOS, Android, and Windows artifacts that the tools parse
- **Tool updates** — when a major feature lands, we'll explain the why and how
- **Community write-ups** — contributors sharing their own forensic research
- **How-to guides** — walkthroughs for getting the most out of iLEAPP, ALEAPP, RLEAPP, VLEAPP, and LAVA

## How to contribute

This blog lives in the [leapps-website repo](https://github.com/abrignoni/leapps-website). To submit a post:

1. Fork the repo
2. Create a new Markdown file in `blog/posts/` using the format `YYYY-MM-DD-your-title.md`
3. Add frontmatter: `title`, `date`, `author`, `tags`, `excerpt`
4. Add your entry to `blog/posts/index.json`
5. Open a pull request

We review all submissions and look forward to seeing what the community has to share.
